 ## Materiales hasta el momento:

* Arduino Uno
* Modulo de reloj DS1302 con libreria <virtuabotixRTC.h>
* Sensor infrarrojo y control remoto
* Servomotor MG995 (Necesita alimentación externa)
* Pantalla LCD  con modulo I2C
